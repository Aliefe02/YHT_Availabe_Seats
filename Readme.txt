(DEPRECIATED)

This program searches for available seats on YHT. YHT stands for High Speed Train in Turkey.

Program automaticly searches for available seats according to user's inputs.
If at least 3 seats are available program sends a whatsapp message to a specific number which was entered by user.

To use this program you need webdriver.

Don't use your computer while running this program.

This program will open and close tabs automaticly. If any fail occurs during loading pages program will reset page.
If you open another tab while program is running than program can't move to the next page after entering information about train.

**IMPORTANT**
    Login your whatsapp web account on your browser before running. Otherwise you will need to login in 5 seconds and program will not send message.
